# Product: Bharat Modules

Bharat Modules is an India-focused B2B industrial procurement platform. It connects verified Indian OEM manufacturers (Class-I, MSME/GST certified) with enterprise and defence buyers for sourcing industrial components.

## Core Value Proposition
- Closed, verified network — no traders or middlemen
- Buyers submit RFQs and receive quotes directly from certified manufacturers
- Manufacturers get listed after a multi-step verification/onboarding process (document upload, M-NDA signing)
- 24-hour SLA on RFQ responses

## User Roles
- **Buyers / OEM Customers** — search catalogue, add to RFQ cart, track orders, manage wishlist, access compliance documents
- **Manufacturers / Vendors** — onboard via verification flow, manage product listings, respond to RFQs via manufacturer portal
- **Platform Admin** — not yet visible in frontend files
